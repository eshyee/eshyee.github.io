查询

单表查询

主要涉及到sq|运算符、单行函数、多行函数、排序、去重

首先创建一个学生表stu, 包含如下字段:

学号id、姓名name、年龄age、性别sex、java成绩score、 体重(kg) weight、 星座xz、 生日birth、 寝室room

向表中插入一条学生信息:

sq|运算符

Like匹配 通配符%

例如:

查询所有姓张的学生

查找姓名中带有“文”字的学生信息

between… and…

在两个值之间

例如:

查找班级里体重在60~70kg的学生姓名

in

is null

值为空，即没有填写值

例如:
查找没有填写星座的学生信息

not

与其他sq|运算符一起使用， 表示取反:

not like not between not in is not null

例如:

查找所有不姓张的学生信息

单行函数

使用规范: 函数名(参数)根据参数, 获取函数的结果

to_date

将字符串转换为日期

例如: to_ date( ‘2018-09-01’ ,’ yyyy-mm-dd’ )

表示将2018-09-01这个字符串，转换为日期数据类型

查询所有1999年出生的学生

to_char

length

获取指定字符串值的长度

例如:

查找所有姓名是两个字的学生信息

查询学生的姓名，以及姓名的长度

concat

拼接两个值，concat(值1,值2)—> 值1值2

查询学生的学号信息(学号信息是指 学号+姓名)

修改stu表,规范星座信息，将所有的星座保存为**座.

查询截止到当前时间每个学生都活了多少天、多少个月

月

天

查询每个学生名字的最后一个字是什么字

多行函数

多行/聚合/分组函数,作用在多行记录上，

avg(字段)字段的平均值

查询班级java成绩的平均分

查询平均体重

查询班级男同学的平均体重

sum(字段)字段求和

查询班级所有学生的体重的和

max(字段)最大值

查询最高分的学生的成绩

查询最重的学生的体重

注意:多行函数，不能出现在where子包含

例如:查询最重的学生的姓名

min(字段)最小值

查询最轻的学生的体重

查询男同学中，最轻的学生的体重

count(*)查询数量

查询表中有多少条记录

查询班里有多少个男生

查询班级里60~ 70分之间的学生有多少个

排序

order by字段 desc

默认从小到大升序，添加desc从大到小降序

查询所有学生信息，并按照体重从小到大排列

查询所有学生信息，按照成绩从大到小排序

去重distinct

一般distinct都用在select查询字段中

分组

查询班级里分别有多少个男生和女生

查询男生和女生的平均分都是多少

查询每个寝室有多少成绩大于80分的同学

通过查询确认一下是否有同年同月同日生的学生

查询每个男生寝室有多少成绩大于80分的同学和名字

分组后的筛选

having

having子句里添加条件,对group by的结果进行条件筛选

having写在group by之后，并且逻辑上也是先得到分组结果，再执行筛选

查询寝室中人数大于5人的那些寝室和人数

查询班里有没有同年同月同日生

查询平均分大于75分的寝室的寝室号和平均分
